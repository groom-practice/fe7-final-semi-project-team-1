# FE7 Final Semi-Project Team 1

FE7 Final Semi-Project Team 1의 프로젝트입니다.

## ✨ 주요 기능 및 기술 스택

- Next.js
- React
- TypeScript
- Tailwind CSS
- React Query
- Axios
- Swiper.js

## 🧑‍💻 기여자 및 담당 페이지

- **이예슬 (팀장):** Login, Posts
- **배근영:** Home, Photos

## 📄 페이지 구성

- **Home:** 메인 페이지 (슬라이드)
- **Photos:** 사진 목록 페이지
- **Login:** 로그인 페이지
- **Posts:** 게시글 페이지

## 📁 파일 구조

- **app:** Next.js의 App Router를 사용한 라우팅 및 페이지 관리
- **components:** 공통적으로 사용되는 UI 컴포넌트
- **lib:** API 호출 등 비즈니스 로직 관련 코드
- **public:** 정적 파일 (이미지 등)
- **types:** TypeScript 타입 정의

## 🚀 시작하기

먼저, 개발 서버를 실행하세요:

```bash
npm run dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000)을 열어 결과를 확인하세요.
